### 🗂️ [Автомонтирование SSHFS папок при входе](/examples/automount-sshfs.md)
Удалённые файловые системы: установка SSHFS, монтирование через fstab/systemd, аутентификация по SSH-ключам, автоматическая настройка скриптом, диагностика проблем

### 🖥️ [Настройка многомониторной конфигурации](/examples/multi-monitor.md)
Несколько мониторов: определение подключённых дисплеев, настройка разрешения и расположения, сохранение конфигурации, масштабирование HiDPI

### 🖨️ [Настройка принтера через CUPS](/examples/printer-setup.md)
Система печати CUPS: установка и запуск, веб-интерфейс, добавление принтера через CLI, драйверы (HP, Canon, Epson, Brother), сетевые принтеры (IPP, AppSocket), диагностика

### 📷 [Подключение сканера](/examples/scanner-setup.md)
Сканирование: установка SANE, проверка обнаружения сканера, настройка через simple-scan/xsane, сетевое сканирование, решение типичных проблем
