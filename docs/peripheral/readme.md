Руководства по настройке периферийного оборудования в РЕД ОС: принтеры, сканеры, серверы печати, смарт-карты, RFID-считыватели и сканеры штрих-кодов.

---

## 📑 Документы раздела

### 🖨️ [Универсальная настройка принтеров](/docs/peripheral/printers-universal.md)
Подключение и настройка принтеров всех основных производителей (HP, Canon, Epson, Brother, Xerox, Samsung, Pantum) через CUPS. Сетевая печать (IPP, Socket, SMB), универсальные драйверы (Gutenprint, Foomatic), автоматический скрипт установки, таблицы совместимости, диагностика.

### 📠 [Настройка сканеров и МФУ](/docs/peripheral/scanners.md)
Архитектура SANE, USB и сетевое сканирование, поддержка HP/Canon/Epson/Brother/Fujitsu, приложения (Simple Scan, XSane, NAPS2), пакетное сканирование, OCR (Tesseract, OCRmyPDF), автоматизация scan-to-PDF.

### 🌐 [Сервер печати для организации](/docs/peripheral/network-printer-server.md)
Развёртывание централизованного сервера печати на базе CUPS + Samba + Avahi. Интеграция с Windows-клиентами, очереди печати, управление пользователями, квоты, мониторинг, Active Directory, GPO, резервное копирование конфигурации.

### 💳 [Смарт-карты и RFID](/docs/peripheral/smartcards-rfid.md)
Работа со смарт-картами и RFID-устройствами: PC/SC Lite, Рутокен, JaCarta, eToken, считыватели ACS ACR122U, аутентификация через PAM, электронная подпись, алгоритмы ГОСТ, интеграция с КриптоПро, настройка браузеров для ЭП, управление сертификатами.

### 📊 [Сканеры штрих-кодов](/docs/peripheral/barcode-scanners.md)
Подключение и настройка сканеров штрих-кодов: режимы HID/COM/OPOS, USB/Bluetooth, интеграция с 1С, форматы (EAN-13, Code 128, QR, DataMatrix), Honeywell/Zebra/Datalogic/Mindeo, генерация штрих-кодов, POS-интеграция, автоматический скрипт настройки.

## 🗺️ Общая архитектура работы с периферией в РЕД ОС

РЕД ОС базируется на RPM-дистрибутиве и использует стандартный стек Linux для работы с периферийным оборудованием:

| Компонент | Назначение | Пакет |
|-----------|-----------|-------|
| **CUPS** | Система печати | `cups`, `cups-filters` |
| **SANE** | Сканирование | `sane-backends`, `sane-frontends` |
| **PC/SC Lite** | Смарт-карты | `pcsc-lite`, `pcsc-lite-ccid` |
| **udev** | Управление устройствами | `systemd-udev` (встроен) |
| **Avahi** | Автоматическое обнаружение | `avahi`, `avahi-tools` |
| **Samba** | Сетевые принтеры (SMB) | `samba`, `samba-client` |

## 🔍 Быстрая диагностика периферии

```bash
#USB-устройства
lsusb

# Посмотреть информацию о принтерах CUPS
lpstat -p -d

# Проверить статус сканера
scanimage -L

# Проверить смарт-карты
pcsc_scan

# Просмотреть системные сообщения о подключении устройств
journalctl -f -u cups
dmesg | tail -30
```
