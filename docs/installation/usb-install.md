## 🛠️ 1. Подготовка ISO-образа

```bash
# Перейти на страницу загрузки в браузере:
# https://redos.red-soft.ru/product/downloads/

# Выбрать версию: РЕД ОС 7.3
# Выбрать редакцию: Desktop или Server
# Нажать «Скачать» — начнётся загрузка ISO-образа

# После загрузки проверить контрольную сумму:
sha256sum ~/Загрузки/REDOS-7.3.1-1.7.3-x86_64.iso

# Сравнить с суммой на сайте (скопируйте значение из раздела «Контрольные суммы»)
echo "abcd1234...  REDOS-7.3.1-1.7.3-x86_64.iso" | sha256sum -c
```

> [!TIP]
> Всегда проверяйте контрольную сумму ISO-образа перед записью — это сэкономит время при диагностике проблем.

## 💾 2. Создание загрузочного накопителя

### 🔹 Ventoy

#### Установка в Windows:

```
1. Скачать Ventoy с https://www.ventoy.net
   - Файл: ventoy-1.x.xx-windows.zip

2. Распаковать архив в любую папку
   - Например: C:\Ventoy\

3. Запустить от имени администратора:
   - Ventoy2Disk.exe

4. Выбрать USB-накопитель в списке

5. Нажать «Install»
   - Подтвердить предупреждение о форматировании

6. Дождаться завершения установки
   - «Install Ventoy Successfully»
```

#### Настройка в Windows:

```
1. После установки флешка будет иметь два раздела:
   - VTOYEFI (скрытый, загрузочный)
   - ventoy (видимый, для ISO-образов)

2. Скопировать ISO-образы простым перетаскиванием:
   - Перетащить REDOS-7.3.1-1.7.3-x86_64.iso в раздел ventoy

3. (Опционально) Создать папки для организации:
   - /ISO/REDOS/
   - /ISO/Ubuntu/
   - /ISO/Windows/
```

#### Создание конфигурации ventoy.json (Windows):

```
1. В разделе ventoy создать папку: \ventoy\config\

2. Создать файл ventoy.json (в Блокноте или редакторе):

{
    "theme": {
        "display_mode": "GUI",
        "resolution": "1920x1080",
        "theme": {
            "title": "Мультизагрузочная флешка - РЕД ОС",
            "footnote": "На основе Ventoy",
            "colors": {
                "normal": "light-gray/blue",
                "selected": "white/red"
            }
        }
    },
    "menu_alias": {
        "REDOS-7.3.1-1.7.3-x86_64.iso": "РЕД ОС 7.3 (Основная)"
    }
}

3. Сохранить файл в кодировке UTF-8 без BOM

4. Извлечь флешку безопасно
```

> [!TIP]
> Ventoy в Windows не требует прав администратора для копирования ISO-образов — только для первоначальной установки.

---

#### Установка в Linux:

```bash
# Скачать Ventoy с https://www.ventoy.net
wget https://github.com/ventoy/Ventoy/releases/latest/download/ventoy-*.tar.gz

# Распаковать
tar -xzf ventoy-*.tar.gz
cd ventoy-*

# Запустить GUI (в Linux)
sudo ./VentoyGUI.x86_64

# Или консольная версия
sudo ./Ventoy2Disk.sh -i /dev/sdX
```

**Преимущества Ventoy:**
- Мультизагрузка — несколько ISO на одном носителе
- Не нужно форматировать при смене образа
- Поддержка образов >4GB (exFAT)
- Работает с UEFI и Legacy BIOS

> [!TIP]
> Скопируйте ISO-образы на флешку простым копированием после установки Ventoy.

## ⚙️ 3. Загрузка с USB

### Вход в Boot Menu

| Производитель | Клавиша |
|---------------|---------|
| ASUS | F8, Esc |
| Acer | F12, F9 |
| Dell | F12 |
| HP | F9, Esc |
| Lenovo | F12, Fn+F12 |
| MSI | F11 |
| Gigabyte | F12 |
| Samsung | Esc, F10 |
| Sony | F11, Assist |
| Toshiba | F12 |

> [!TIP]
> Для UEFI-загрузки выбирайте пункт с пометкой «UEFI: <название флешки>».

---

### Настройка BIOS/UEFI

```
1. Отключить Secure Boot (опционально)
2. Включить Legacy/CSM (если нужна поддержка BIOS)
3. Установить приоритет загрузки с USB
4. Сохранить настройки (F10)
```

> [!NOTE]
> РЕД ОС 7.3 поддерживает UEFI Secure Boot, но для сторонних драйверов может потребоваться отключение.

## 🖥️ 4. Установка системы

### Основные этапы:

```
1. Выбор языка → Русский
2. Раскладка клавиатуры → Russian
3. Разметка диска:
   - Автоматически (рекомендуется)
   - Вручную (для опытных)
4. Выбор диска для установки
5. Настройка пользователя и пароля
6. Начало установки (15-30 минут)
7. Перезагрузка
```

### Рекомендуемая разметка (для ручной настройки):

| Точка монтирования | Размер | Файловая система |
|--------------------|--------|------------------|
| `/boot/efi` | 512 MB | FAT32 (для UEFI) |
| `/boot` | 1 GB | ext4 |
| `/` | 50-100 GB | ext4 |
| `/home` | Остальное | ext4 |
| `swap` | = RAM (или 4-8 GB) | swap |

> [!TIP]
> Для SSD выбирайте файловую систему ext4 с опциями `noatime,discard`.

## 🔍 5. Диагностика проблем

### Система не загружается с USB

```bash
# Проверить целостность ISO
isoinfo -d -i REDOS-7.3.1-1.7.3-x86_64.iso

# Перезаписать флешку (Ventoy)
sudo ./Ventoy2Disk.sh -i /dev/sdX

# Проверить флешку на ошибки
badblocks -sv /dev/sdX
```

> [!NOTE]
> Частая причина — неправильная запись образа или повреждённый ISO.

---

### Ошибки при установке

```bash
# Проверить оперативную память
memtest86+

# Проверить диск
sudo smartctl -a /dev/sda

# Просмотр логов установки
Ctrl+Alt+F2  # переключение в консоль
cat /tmp/anaconda.log
```

> [!NOTE]
> Ошибки «Input/output error» часто указывают на проблемы с носителем.

---

### Чёрный экран после загрузки

```bash
# Добавить параметр ядра при загрузке:
nomodeset

# Для видеокарт NVIDIA:
nouveau.modeset=0

# Для видеокарт AMD:
radeon.modeset=0
```

> [!TIP]
> Нажмите `e` в меню загрузчика для редактирования параметров ядра.

## 🎯 6. Лайфхаки


### 🔹 Быстрая проверка ISO перед записью

```bash
# Проверка на наличие загрузочной записи
isoinfo -d -i REDOS-7.3.1-1.7.3-x86_64.iso | grep -i boot

# Проверка размера (должен быть >2GB)
ls -lh REDOS-7.3.1-1.7.3-x86_64.iso

# Монтирование для проверки содержимого
sudo mount -o loop REDOS-7.3.1-1.7.3-x86_64.iso /mnt
ls /mnt
sudo umount /mnt
```

> [!TIP]
> Если `isoinfo` возвращает ошибку — образ повреждён или не является загрузочным.

### 🔹 Сохранение настроек Ventoy для будущих установок

```bash
# Резервное копирование конфигурации Ventoy
cp /mnt/ventoy/ventoy.json ~/backup-ventoy.json

# Восстановление на новой флешке
cp ~/backup-ventoy.json /mnt/ventoy/ventoy.json
```

> [!TIP]
> Сохраните `ventoy.json` в облако — пригодится при создании новой флешки.

### 🔹 Запись с прогресс-баром (альтернатива dd)

```bash
# Использовать pv для отображения прогресса
pv REDOS-7.3.1-1.7.3-x86_64.iso | sudo dd of=/dev/sdX bs=4M
```


> [!TIP]

### 🔹 Резервное копирование текущей системы для быстрой установки

```bash
# Создать образ текущей системы
sudo dd if=/dev/sda of=~/system-backup.img bs=4M status=progress

# Сжать образ
gzip ~/system-backup.img

# Восстановление
gunzip -c ~/system-backup.img.gz | sudo dd of=/dev/sda bs=4M status=progress
```

> [!WARNING]
> Требует места, равного размеру диска. Используйте для клонирования одинаковых систем.

## 🤖 7. Автоматизация подготовки

Для автоматической подготовки загрузочной флешки с Ventoy используйте скрипт `usb-install.sh`.

### 📥 Запуск скрипта

```bash
# Скачать и выполнить скрипт
curl -sL https://github.com/teanrus/redos-lifehacks/releases/latest/download/usb-install.sh | sudo bash

# Или скачать и запустить вручную
wget https://github.com/teanrus/redos-lifehacks/releases/latest/download/usb-install.sh
chmod +x usb-install.sh
sudo ./usb-install.sh
```

> [!IMPORTANT]
> Скрипт требует прав root. Все данные с флешки будут удалены!

### 🔧 Описание работы

Скрипт выполняет следующие действия:

1. **Поиск USB-накопителей**
   - Отображает все подключённые флешки
   - Показывает размер и модель
   - Запрашивает выбор устройства

2. **Проверка ISO-образа**
   - Проверяет наличие ISO в текущей директории и в папке «Загрузки»
   - Предлагает скачать с официального сайта
   - Проверяет контрольную сумму SHA256

3. **Установка Ventoy**
   - Скачивает последнюю версию Ventoy
   - Устанавливает на выбранную флешку
   - Копирует ISO-образ

4. **Настройка конфигурации**
   - Создаёт `ventoy.json` с кастомным меню
   - Добавляет алиасы для образов
   - Настраивает тему оформления

5. **Проверка результата**
   - Проверяет загрузочную запись
   - Выводит информацию о флешке
   - Показывает полезные команды

### ⚙️ Параметры скрипта

| Параметр | Описание |
|----------|----------|
| `USB_DEVICE` | Устройство флешки (например, `/dev/sdb`) |
| `ISO_PATH` | Путь к ISO-образу РЕД ОС |
| `ISO_DOWNLOAD_PAGE` | Ссылка на страницу загрузки РЕД ОС |
| `ISO_FILENAME` | Имя файла ISO-образа |
| `ISO_EXPECTED_SHA256` | Ожидаемая контрольная сумма SHA256 |
| `VENTOY_VERSION` | Версия Ventoy для загрузки |
| `CREATE_CONFIG` | Создавать ли `ventoy.json` |

> [!NOTE]
> Все параметры задаются интерактивно в ходе выполнения скрипта. ISO-образ загружается вручную с официального сайта. Каждое действие требует подтверждения пользователя.

### 🔍 Диагностика

Если скрипт не работает:

```bash
# Проверить подключённые флешки
lsblk -o NAME,SIZE,MODEL,TYPE

# Проверить точку монтирования
mount | grep /dev/sd

# Проверить права доступа
ls -la /dev/sdX

# Проверить Ventoy вручную
cd ventoy-*
sudo ./Ventoy2Disk.sh -l  # список устройств
sudo ./Ventoy2Disk.sh -i /dev/sdX  # установка
```
