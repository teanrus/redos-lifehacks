## 📚 Лайфхаки по установке офисных пакетов

### 1. Использование DNF вместо YUM

В РЕД ОС используется современный пакетный менеджер **DNF** (Dandified YUM):

```bash
# Правильно (DNF):
dnf install r7-office
dnf update r7-office
dnf clean all

# Неправильно (YUM - устарело):
yum install r7-office  # не рекомендуется
```

> **Зачем:** DNF работает быстрее, лучше разрешает зависимости и потребляет меньше памяти.

---

### 2. Оптимизация DNF для быстрой загрузки

Добавьте в `/etc/dnf/dnf.conf` для ускорения загрузки пакетов:

```bash
# Резервное копирование
cp /etc/dnf/dnf.conf /etc/dnf/dnf.conf.backup

# Добавление оптимизаций
echo "max_parallel_downloads=10" >> /etc/dnf/dnf.conf
echo "fastestmirror=True" >> /etc/dnf/dnf.conf
echo "deltarpm=True" >> /etc/dnf/dnf.conf
```

| Параметр | Описание | Эффект |
|----------|----------|--------|
| `max_parallel_downloads=10` | Параллельная загрузка пакетов | Ускорение в 3-5 раз |
| `fastestmirror=True` | Выбор быстрого зеркала | Быстрее загрузка метаданных |
| `deltarpm=True` | Загрузка дельта-пакетов | Экономия трафика до 70% |

> **Зачем:** Ускорение установки и обновления пакетов.

---

### 3. Установка Р7-Офис через репозиторий

Вместо ручного скачивания RPM используйте официальный репозиторий:

```bash
# Установка репозитория
dnf install -y r7-release

# Обновление кэша
dnf makecache

# Установка Р7-Офис
dnf install -y r7-office
```

> **Зачем:** Автоматические обновления через `dnf update r7-office`.

---

### 4. Проверка установленных офисных пакетов

```bash
# Список всех офисных пакетов
rpm -qa | grep -E 'libreoffice|r7-office|myoffice'

# Версия конкретного пакета
rpm -q r7-office

# Список репозиториев
dnf repolist
```

> **Зачем:** Быстрая проверка установленных пакетов и репозиториев.

---

### 5. Резервное копирование настроек DNF

Перед изменением конфигурации DNF создайте резервную копию:

```bash
cp /etc/dnf/dnf.conf /etc/dnf/dnf.conf.backup.$(date +%Y%m%d_%H%M%S)
```

> **Зачем:** Возможность быстрого отката при ошибке в настройках.

---

### 6. Установка нескольких офисных пакетов параллельно

РЕД ОС позволяет установить несколько офисных пакетов одновременно:

```bash
# Установка всех пакетов сразу
dnf install -y libreoffice r7-office

# Проверка установки
rpm -q libreoffice r7-office
```

> **Зачем:** Разные пакеты для разных задач (совместимость, криптография, базовые нужды).

---

### 7. Настройка офисного пакета по умолчанию

```bash
# Установить Р7-Офис по умолчанию для документов Word
xdg-mime default r7-office-writer.desktop application/msword

# Установить LibreOffice по умолчанию для таблиц Excel
xdg-mime default libreoffice-calc.desktop application/vnd.ms-excel

# Проверка настроек
xdg-mime query default application/msword
```

> **Зачем:** Открытие файлов в нужной программе по двойному клику.

---

### 8. Очистка кэша при проблемах с обновлением

При ошибках обновления очистите кэш DNF:

```bash
# Полная очистка кэша
dnf clean all
rm -rf /var/cache/yum /var/cache/dnf

# Пересоздание кэша
dnf makecache

# Обновление пакета
dnf update -y r7-office
```

> **Зачем:** Решение проблем с устаревшими метаданными репозиториев.

---

### 9. Активация лицензии Р7-Офис без GUI

Для серверов или минималистичных установок:

```bash
# Создание директории для лицензии
sudo mkdir -p /etc/r7-office/license

# Копирование файла лицензии
sudo cp license.lickey /etc/r7-office/license/

# Проверка прав (должны быть на чтение и запись!)
sudo chmod 644 /etc/r7-office/license/license.lickey
```

> **Зачем:** Активация на серверах без графического интерфейса.

### 10. Установка МойОфис (бесплатно для частных лиц)

**МойОфис** имеет бесплатную версию для домашних пользователей:

```bash
# Скачайте RPM-пакет с официального сайта
wget https://myoffice.ru/products/download/desktop/

# Установка через DNF (с автоматическим разрешением зависимостей)
sudo dnf install -y ./myoffice-documents-desktop*.rpm

# При ошибках зависимостей
sudo dnf install -f ./myoffice-documents-desktop*.rpm
```

**Особенности МойОфис:**
- ✅ Бесплатная версия для частных лиц
- ✅ Поддержка СКЗИ через КриптоПро CSP
- ✅ Сертификат ФСТЭК
- ✅ В реестре отечественного ПО
- ✅ Не требует регистрации для домашней версии

> **Зачем:** Отечественное решение с поддержкой российской криптографии.

| **В реестре отечественного ПО** | ❌ | ✅ | ✅ |

> **Выбор:** Для госсектора — **Р7-Офис** или **МойОфис**. Для дома — **МойОфис (бесплатно)** или **LibreOffice**.

## ⚠️ Важные замечания

1. **Лицензия Р7-Офис**: Для полноценной работы требуется коммерческая лицензия.
2. **Активация**: Поместите файл лицензии в `/etc/r7-office/license/license.lickey`.
3. **Права на файл лицензии**: Файл должен иметь права **на чтение и запись** (не только чтение!).
4. **Несколько пакетов**: Можно установить несколько офисных пакетов параллельно.
5. **Настройка по умолчанию**: Используйте `xdg-mime` для выбора программы по умолчанию.
6. **МойОфис**: Бесплатная версия для частных лиц не требует регистрации.
7. **КриптоПро CSP**: Требуется для работы с криптографией в МойОфис.
