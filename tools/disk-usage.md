## Автоматизация

### Скрипт disk-usage.sh

Для автоматического выполнения всех описанных ниже действий используйте скрипт [`disk-usage.sh`](https://github.com/teanrus/redos-lifehacks/blob/main/disk-usage.sh).

**Быстрый запуск:**

```bash
curl -sL https://github.com/teanrus/redos-lifehacks/releases/latest/download/disk-usage.sh | sudo bash
```

**Или вручную:**

```bash
wget https://github.com/teanrus/redos-lifehacks/releases/latest/download/disk-usage.sh
chmod +x disk-usage.sh
sudo ./disk-usage.sh
```

**Возможности скрипта:**

- Интерактивное меню с выбором действий
- Быстрая диагностика дисков (`df`)
- Поиск больших файлов (>1ГБ)
- Топ-10 самых больших файлов
- Анализ занятия каталогов (`du`)
- Очистка кэша пакетного менеджера (APT/DNF)
- Очистка системных журналов
- Поиск дубликатов файлов (требуется `fdupes`/`rdfind`)
- Проверка удалённых, но занятых файлов
- Генерация полного отчёта в файл

## Быстрая диагностика

### Общая информация о дисках

```bash
df -h
```

Показывает использование всех подключённых файловых систем в человеко-читаемом формате.

```bash
df -hT
```

Отображает дополнительную информацию о типе файловой системы (ext4, xfs и т.д.).

```bash
df -i
```

Показывает использование inode — полезно при проблемах с множеством мелких файлов.

### Сводка по корневому разделу

```bash
df -h /
```

Быстрая проверка свободного места на системном разделе.

## Поиск больших файлов

### Файлы размером более 1 ГБ

```bash
sudo find / -type f -size +1G -exec ls -lh {} \; 2>/dev/null | awk '{print $5, $9}'
```

Находит все файлы размером более 1 ГБ и выводит их размер и путь.

### Топ-10 самых больших файлов

```bash
sudo find / -type f -exec du -h {} \; 2>/dev/null | sort -rh | head -10
```

Показывает 10 самых больших файлов в системе.

### Файлы, не изменявшиеся более года

```bash
sudo find /home -type f -mtime +365 -exec ls -lh {} \; 2>/dev/null
```

Поиск файлов, которые не изменялись более 365 дней (может указывать на ненужные данные).

## Анализ занятия каталогов

### Использование du для анализа папок

```bash
du -sh /* 2>/dev/null | sort -rh
```

Показывает размер всех каталогов в корне, отсортированных по убыванию.

### Глубина анализа 2 уровня

```bash
du -h --max-depth=2 /home 2>/dev/null | sort -rh
```

Анализ занятия домашнего каталога с детализацией до 2 уровней вложенности.

### Интерактивный анализ с ncdu

```bash
sudo apt install ncdu
sudo ncdu /
```

**ncdu** — интерактивная утилита для навигации по дисковому пространству с возможностью удаления файлов прямо из интерфейса.

**Горячие клавиши ncdu:**

- `↑`/`↓` — навигация
- `Enter` — войти в каталог
- `d` — удалить файл/каталог
- `q` — выход

## Очистка кэша пакетного менеджера

### Очистка кэша APT

```bash
sudo apt clean
```

Полная очистка кэша загруженных пакетов.

```bash
sudo apt autoclean
```

Удаление только устаревших версий пакетов из кэша.

```bash
sudo apt autoremove
```

Удаление ненужных пакетов-зависимостей, установленных автоматически.

### Проверка размера кэша

```bash
du -sh /var/cache/apt/archives
```

Показывает текущий размер кэша пакетов APT.

### Очистка журналов системы

```bash
sudo journalctl --vacuum-time=7d
```

Удаление логов старше 7 дней.

```bash
sudo journalctl --vacuum-size=100M
```

Ограничение размера журналов до 100 МБ.

## Поиск дубликатов файлов

### С помощью fdupes

```bash
sudo apt install fdupes
fdupes -r /home/username
```

Поиск дубликатов в домашнем каталоге.

```bash
fdupes -r -d /home/username
```

Интерактивное удаление дубликатов (оставляет один экземпляр).

### С помощью rdfind

```bash
sudo apt install rdfind
rdfind -dryrun true /home/username
```

Поиск дубликатов без удаления (тестовый режим).

```bash
rdfind -makesymlinks true /home/username
```

Замена дубликатов символическими ссылками.

## Мониторинг в реальном времени

### Отслеживание изменений в реальном времени

```bash
watch -n 5 'df -h /'
```

Обновление информации о свободном месте каждые 5 секунд.

### Логирование использования диска

```bash
while true; do echo "$(date): $(df -h / | tail -1)" >> /var/log/disk-usage.log; sleep 3600; done &
```

Запись статистики использования диска в лог каждый час (в фоновом режиме).

### Проверка занятия конкретным процессом

```bash
sudo lsof +L1
```

Показывает файлы, удалённые, но всё ещё занимающие место (удерживаются процессами).

```bash
sudo find /proc/*/fd -lname '/var/log' -ls 2>/dev/null
```

Поиск процессов, удерживающих удалённые файлы логов.

**Решение:** перезапустить соответствующий сервис или отправить сигнал `SIGHUP`:

```bash
sudo kill -HUP $(pidof rsyslog)
```
