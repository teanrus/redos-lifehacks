  <summary>Очистка системы от временных файлов (cleanup.sh)</summary>
  
### 📝 Описание скрипта
- Анализ дискового пространства — показывает состояние до и после очистки
- Очистка временных файлов:
    - /tmp — удаляет файлы, не использовавшиеся более 1 дня
    - /var/tmp — удаляет файлы, не использовавшиеся более 7 дней
- Очистка системного кэша:
    - Кэш DNF (пакеты, метаданные)
    - Системные журналы (journalctl) — оставляет только последние 7 дней
- Очистка логов:
    - Удаляет старые логи старше 30 дней
    - Удаляет пустые лог-файлы
- Очистка пользовательских данных:
    - Кэш браузеров (Яндекс.Браузер, Chromium)
    - Кэш мессенджеров (Telegram, Viber)
    - Временные файлы пользователя (~/.cache, ~/.thumbnails)
- Удаление неиспользуемых пакетов:
    - Старые ядра (оставляет последние 2)
    - Неиспользуемые зависимости (dnf autoremove)
    - Кэш PackageKit
    - Очистка корзины — удаляет файлы из корзины пользователя
    - Очистка старых бэкапов — удаляет бэкапы конфигов старше 90 дней

### Безопасность:
- Каждое действие требует подтверждения — вы сами выбираете, что удалять
- Показывает размер директорий перед очисткой
- Сохраняет последние 2 ядра — на случай проблем с новым
- Сохраняет журналы за 7 дней — для диагностики проблем
- Безопасное удаление через rm -rf с проверкой существования

### Использование:
```bash
# Скачать скрипт
wget https://raw.githubusercontent.com/teanrus/redos-lifehacks/main/scripts/utils/cleanup.sh
# Сделать исполняемым
chmod +x cleanup.sh
# Запустить с правами root
sudo ./cleanup.sh
```
